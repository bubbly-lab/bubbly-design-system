# Bubbly Design System

버블리랩의 프로덕트를 위한 디자인 시스템입니다.

## 패키지 구조

| 패키지 | 설명 |
|---|---|
| [`design-tokens`](./packages/design-tokens) | DTCG JSON 디자인 토큰 정의 + 빌드 |
| [`react`](./packages/react) | Panda CSS + Ark UI 기반 React 컴포넌트 (준비 중) |

## 시작하기

```bash
pnpm install
pnpm build
```
